/DOCKER MEDIA LAB/

Implementación de un entorno de servidor doméstico (HomeLab) utilizando Docker y Docker Compose. El objetivo fue centralizar servicios de streaming multimedia y gestión de archivos en un contenedor ligero, garantizando la portabilidad y la facilidad de despliegue mediante infraestructura como código.

Ver repositorio GitHub

Logré automatizar el despliegue de Jellyfin, Sonarr y un servidor de archivos, reduciendo el tiempo de configuración de horas a minutos. Este proyecto me permitió dominar redes internas de Docker, volúmenes persistentes y la gestión de permisos en entornos Linux (Bash).
