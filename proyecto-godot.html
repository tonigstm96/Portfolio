/GODOT GUNSMITH/

Desarrollo de un prototipo técnico en Godot Engine enfocado en mecánicas de inventario modular. El reto principal fue crear un sistema de personalización de armas donde cada componente (cañón, mira, culata) modifica las variables de comportamiento del objeto mediante scripts en GDScript.

Ver repositorio GitHub

Apliqué patrones de diseño como el "Componente" y el "Estado" para gestionar las animaciones y la lógica de disparo. Fue mi primera incursión seria en el desarrollo de software orientado a videojuegos, fusionando lógica matemática con feedback visual en tiempo real.
